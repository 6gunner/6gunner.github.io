React-svg-loader的使用

  • 在nuxt项目里

安装依赖： @nuxtjs/svg

配置nuxt.js:

引入svg图片，类型为raw:

import HBLogo from "~/assets/Huobi.svg?raw"; 

export default {
  name: "Investors",
  data() {
    return {    
      imgList: [
        HBLogo,
      ]
    }
  }
}

在html里使用svg图片，作为html标签代码

<div class="imgDiv" v-for="(item, index) of imgList" :key="index">
  <div v-html="item" class="svg"></div>
</div>
