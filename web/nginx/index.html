    # nginx的进程管理模块

    nginx的进程模型有2种：

    Single是单进程模型

    Master是master-worker进程模型

    master在服务启动时，进行全局初始化，然后管理worker进程。

    master进程负责接受请求，worker进程负责处理请求。
