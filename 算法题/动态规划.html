          # 什么是动态规划？

          动态规划的算法，都是先从递归，然后演变成记忆化搜索，最后演变成动态规划的。

          一般动态规划都是有固定套路：

          1.找可变参数

          2.将可变参数映射成一张表

          3.标记一下答案的位置

          4.根据base case, 来填写表里面不需要依赖其他位置的值

          5.==分析其他位置的值怎么计算得到==

          6.填表，返回最终答案。

          网上找到一个视频，关于找零钱的方法

          dp[0][k*arr[0]] = 1， k满足条件aim % arr[0] = 0

          在动态规划算法里，最核心的就是==通用公式怎么推导==

          这道题dp[i][j]的值 等于

          dp[i][j] = dp[i-1][j] + dp[i-1][j-arr[i]] + dp[i-1][j-arr[i]*2] +...+ dp[i][j-arr[i]*k]

当j - arr[i] * k >= 0 即可

            if (i >= coins[j]) {
                let k = Math.floor(i / coins[j]);
                if (i % coins[j] == 0) {
                    dp[i] = Math.min(dp[i], k);
                } else if (dp[i-coins[j]] != -1) {
                    dp[i] = Math.min(dp[i], 1 + dp[i-coins[j]]);
                }
            }
        }
        dp[i] = dp[i] == Number.MAX_SAFE_INTEGER ? -1 : dp[i];
    }
    return dp[amount];
};

          优化后的代码

          /**
 * @param {number[]} coins
 * @param {number} amount
 * @return {number}
 */
// 用来保存最小的结果，dp[1]表示用coins里的硬币，换1块钱最少的方式
const dp = [];
var coinChange = function(coins, amount) {
  	// 可以不排序
    let sortCoins = Array.prototype.sort.call(coins, (n1, n2) => {
        return n1 - n2;
    });
    dp[0] = 0;
    for (let i = 1; i <= amount; i++) {
        // 定义一个最小值
        let min = Number.MAX_VALUE;
        for (let j = 0; j < sortCoins.length; j++) {
          	// 用coins的硬币，换 i-sortCoins[j] 后剩余硬币最小数量。
            if (i >= coins[j] && dp[i - sortCoins[j]] < min) {
                min = Math.min(dp[i - sortCoins[j]] + 1, min);
            }
        }
        dp[i] = min;
    }
    // console.log(dp);
    return dp[amount] == Number.MAX_VALUE ? -1 : dp[amount];
}

          # 例题3 回文

          https://leetcode-cn.com/problems/longest-palindromic-substring/

          暴力解法

          利用广度优先的算法，将str分成二叉树，(0, str-1) (1, str) 一次去寻找

          /**
 * @param {string} s
 * @return {string}
 */
var longestPalindrome = function(s) {

    function isPalindrome(str) {
      	// 根据中心判断是否对称
        let length = str.length;
        if (length % 2 == 0) {
            return str.slice(0, length / 2) == str.slice(length/2, length);
        } else {
            return str.slice(0, length / 2) == str.slice(length/2 + 1, length);
        }
    }

    if (!s.length) {
        return '';
    }
    if (s.length == 1) {
        return s;
    }

    let result = "";
    const array = [];
    array.push(s);
    while (array.length) {
        let str = array.pop();
        if (isPalindrome(str)) {
            result =  str;
            break;
        }
        array.unshift(str.substr(0, str.length-1));
        array.unshift(str.substr(1, str.length));
    }
    return result;
};

          ==正确思路==

          动态规划

          回文串的特点是字符串无论是正读还是反读，都是一样的顺序。

          当一个字符串是回文串时，在字符串两端加上两个同样的字符串，那么字符串依然还是回文串。

          反之，如果一个字符串本身不是回文串，那么无论两端加上什么，都无法是回文串。

          所以，这道题用动态规划解的思路就是去判断字符串去掉收尾之后，是否依然是回文串

          用dp[i][j]代表从i截取到j是否是回文串，那么dp[i][j]=dp[i+1][j-1] && s[i]== s[j]。

          我的答案

          /**
 * @param {string} s
 * @return {string}
 */
var longestPalindrome = function(s) {

    function isPalindrome(str) {
        let length = str.length;
        if (length % 2 == 0) {
            return str.slice(0, length / 2) == str.slice(length/2, length);
        } else {
            return str.slice(0, length / 2) == str.slice(length/2 + 1, length);
        }
    }

    if (!s.length) {
        return '';
    }
    if (s.length == 1) {
        return s;
    }
    let i = 0; // 指针1 在→
    let j = 0; // 指针2 在←
    const sArray = s.split('');
    // dp[i][j]表示从j到i是回文串
    const dp = [];
    // max表示从max[1]到max[0]是最大长度
    let max = [0, 0];
    for (i = 0; i <= s.length; i++) {
        dp[i] = [];
        dp[i][i] = true;
        if (i > 0) {
            dp[i][i-1] = true;
            if (max[0] - max[1] < 1) {
                max = [i, i-1];
            }
        } 
        if (i > 1) {   
            for (j = 0; j < i - 1; j ++) {
                dp[i][j] = dp[i-1][j+1] && sArray[i-1] == sArray[j]; // 首尾相同
                // console.log(`dp[${i}][${j}] = ${dp[i][j]}`)
                if (dp[i][j] && i - j > max[0] - max[1]) {
                    max = [i, j];
                }
            }
        }
    }
    // 最后一次性截取
    return s.slice(max[1], max[0])
};
