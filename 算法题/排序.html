        # 冒泡排序

        1、外层 for 循环控制循环次数

        2、内层 for 循环进行两数交换，找每次的最大数，排到最后。

        3、循环第2步，直到遍历结束

        4、可以设置一个标志位，减少不必要的循环。

        function bubbleSort(sds) {
for (let i = 0; i < sds.length - 1; i++) {
    for (let j = 0; j < sds.length - 1 - i; j++) {
      if (sds[j] > sds[j + 1] && sds[j] != "null" && sds[j + 1] != "null") {
        let temp = sds[j];
        sds[j] = sds[j + 1];
        sds[j + 1] = temp;
      }
    }
  }
}

        # 选择排序

        1、在无序数组里，遍历n-1个数，找到最小的数，和第一个数进行交换。

        2、遍历n-2个数，找到最小的，和第二数位置交换。

        3、重复以上操作直到第 n-1 次遍历最小的数和第 n-1 个数交换，排序完成。

        动图

        代码

        function selectSort(arr) {
  function swap(left, right) {
    let temp = arr[left];
    arr[left] = arr[right];
    arr[right] = temp;
  }
  for (let i = 0; i < arr.length; i++) {
    let min = i ;
    for ( let j = i; j < arr.length; j++) {
      if (arr[j] < arr[min]) {
        min = j
      }
    }
    swap(i, min);
  }
  return arr;
}

const nums = [3, 44, 38, 5,  47, 15, 36, 26 , 27,  2,46,4,19,50,48];

        # 快速排序

        核心思想

        选定一个中心位置 pivot，将数组中比pivot大的数移动到pivot右边，比pivot小的数移动到左边。

        步骤

        1、将数组最左边的数设置为中心位置，设定两个指针（left、right）。从右侧第0个位置开始，如果数字大于pivot，将right指针减1；如果数字小于 pivot，则将数字移动到left指针的位置上，将left指针 + 1；

        2、接着，将left指针的数和pivot比较，如果数值比pivot小，left + 1；如果比pivot大，将值赋给right指针上，将right指针+1。

        3、当left和right重合，将pivot赋值给该位置上的元素。

        代码

        function quickSort(arr) {
  if (!arr.length || arr.length == 1) {
    return arr;
  }
  function _quickSort(start, end) {
    if (start >= end) {
      return;
    }
    let left = start;
    let right = end;
    let privot = arr[left];
    while(left < right) {
      while (arr[right] > privot && right > left ) {
        right --;
      }
      if (right > left) {
        arr[left] = arr[right];
        left++;
      }
      while (arr[left] < privot && right > left) {
        left++;
      }
      if (right > left) {
        arr[right] = arr[left];
        right --;
      }
    }
    // 循环结束
    if (left == right) {
      arr[left] = privot;
      _quickSort(start, left - 1);
      _quickSort(left + 1, end);
    }
  }
  _quickSort(0, arr.length - 1);
  return arr;
}

        # 例题2

        # 把数组排成最小的数

        我的思路

        • 将x，y两个数转换为字符串，从左边开始比较，如果位置数不相等，

          • x[i] < y [i], 则 x < y;
          • x[i] > y [i], 则 x > y;

        • 如果两个数，长度不一样，但是前面的几位数一直相等，那么截取长度较长的那个数，将剩下来的数和长度较短的数进行比较。

        比如 121， 12 。 因为 12都一样，那么将 121剩下的1和12进行比较，再将1和2进行比较。 得出 1 < 2 ，所以 121 < 12。

        /**
 * @param {number[]} nums
 * @return {string}
 */
var minNumber = function(nums) {
    if (!nums.length) {
        return "";
    }
    function sort(a, b) {
        // console.log(`a = ${a}, b = ${b}`);
        if (a == b) {
            return 0;
        }
        if (a < 10 && b < 10) {
            return a - b;
        } else {
           let aStr = a.toString();
           let bStr = b.toString();
           let length = aStr.length < bStr.length ? aStr.length : bStr.length;
           for (let i = 0; i < length; i++) {
                let aLetter = aStr.substr(i, 1);
                let bLetter =bStr.substr(i, 1);
                // console.log(`aLetter = ${aLetter}, bLetter = ${bLetter}`);
                if (aLetter != bLetter) {
                   return aLetter - bLetter;
                }
           }
           if (aStr.length > length) {
               return sort(aStr.slice(length, aStr.length), b);
           } else {
               return sort(a, bStr.slice(length, bStr.length))
           }
        }
    }
    nums.sort(sort);
    return nums.join("");
};

        官方解题思路：

        • 此题求拼接起来的 “最小数字” ，本质上是一个排序问题。
        • 排序判断规则： 设 numsnums 任意两数字的字符串格式 x 和 y ，则
          • 若拼接字符串 x + y > y + x ，则 m > n；
          • 反之，若 x + y < y + x ，则 n < m； 根据以上规则，套用任何排序方法对 nums执行排序即可。
